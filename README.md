## 💡 Features (Implemented in `Doctor.java`)

- **View All Doctors**:  
  Displays a formatted table of all doctors with their ID, Name, and Specialization.

- **Search Doctor by ID**:  
  Check whether a doctor with a specific ID exists in the database.
